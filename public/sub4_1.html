솔루션(PLIMS, Project Life Cycle Information Management System)의 강점 및 차별화 포인트

1. 다수의 중·대형 및 차세대 규모 프로젝트 관리자 요청 기반 현장 중심 기능 구성 및 적용 검증
2. 시스템 13개 영역 기능 모듈화로 기업 규모/필요 기반 기능 선택 및 도입 가능
3. Project Data 기반 PM 관리 Point별 Dash-board를 통한 Real Time Status 정보 제공
4. PM 의사결정 및 주요 이해관계자 성과 공유를 위한 계획 대비 성과 Visualization 강화
5. 국내외 SW 품질 모델 기반 Eng. 및 관리 영역 권장 실무 Practice Coverage 확보
6. 프로젝트 데이터 Import/Export 기능을 통한 Data 입력 용이성 및 데이터 활용성 제고

주요 기능 - Status 정보 및 사업관리

기능 개요

정량적 Data를 기반으로 프로젝트 L/C(계획 및 착수, 수행 및 조정, 완료) 전 기간에 대해 계획 대비 성과를 요약하여 Dash-board를 제공합니다.

PM의 활용

  • 프로젝트 계획 대비 성과 Data 기반 개관적 Status 인지
  • Data 기반 고위험 시점 사전 예측 및 의사결정 대응

세부 기능 및 강점

  • Status 요약 현황
  • 계획 대비 성과
  • 요구사항 현황(변경, 상태)
  • Action Item 관리 현황
  • Issue 관리 현황
  • Risk 및 기회관리 현황
  • 공자/수량
  • 프로젝트 행사/이벤트 관리

주요 메뉴 구성

정량적 프로젝트 현황(7)
  • 프로젝트 주요 관리현황
  • Status 요약 현황
  • 계획 대비 성과(By 기준일)
  • 요구사항 변동/Status 및 변경 현황
  • Action Item 관리 현황
  • Issue 관리 현황
  • 위험 및 기회관리 현황
사업 수행 관리(13)
  • 인적 자원 관리 / 사업범위·이슈·위험 기회관리
  • 인적자원 대가관리 / Action Item 관리
  • 인적자원 계획 / 이슈(Issue) 관리
  • 인적자원 투입실적 관리 / 리스크(Risk) 관리
  • 인적자원 투입 일정 / 요구사항 변경관리
  • 인적자원 투입 현황(누적) / 형상통합 관리
  • 인적자원 투입 상세 / 형상통보 변경관리

주요 기능 - 계획수립 및 진척관리 개요

기능 개요

프로젝트 일정계획 및 진척관리를 위해 총 11개 기능으로 구성합니다.

PM의 활용

  • 전체 프로젝트 범위/일정/인력을 계획하고 수행 현황 확인
  • 고정형뿐 아니라 미래 일정 변경 예측 가능
  • Critical Path 확인 기반 핵심 Task 관리
  • 단계별 마일스톤 기반 Task 조정
  • WBS Top Down/Bottom Up 방식 계획 작성
  • 담당자 실적 입력 후 즉시 승인 및 현황처리
  • 기 유사 WBS를 재사용해 작업 등록 가능

세부 기능 및 강점

  • WBS 작업 화면 또는 엑셀 대량 작업/작성이 용이
  • MSP 연계 구조로 네트형 활동을 Gantt Chart로 표현
  • 일정 변경 시 진척 계산이 실시간 집계됨
  • 월/주/일 주기별 계획 대비 실적 정보 제공
  • SW Engineering 요구사항·설계/개발/테스트·품질 실적 연계

주요 메뉴 구성

Task 테일러링 및 작성
  • WBS 항목/범주 생성 및 정의
  • 유사 사업기반 Task 작성
  • Task 진행 결과 조회
일정계획 & 진척관리
  • 상세 Task 계획 일정계획 수립
  • 업무영역별 진척관리
  • 일간 진척산정
  • 일별 진척산정 결과 조회
수행 실적관리
  • 수행 실적 일계(By 담당자)
  • 실적 변경 및 승인(By PM)
  • 실적 조정 및 편차(PM/기준일)
  • 수행 실적 조회(By 기준일)

주요 기능 - SW 엔지니어링 활동 관리 개요

기능 개요

소프트웨어 생명주기(분석/설계/개발/테스트/품질) 수행 내역을 등록하고 계획 및 성과(수행 진척)를 관리합니다.

PM의 활용

  • 개발 제품 전체 화면 목록 관리로 필수 관리/수행 현황 확인
  • 개발 제품 전체 프로그램 목록 관리로 필수 관리/수행 현황 확인
  • 단위·통합 테스트 케이스를 통해 테스트 진척 추적
  • 제품 품질 확보를 위해 사출 결함 이슈 등록(FMEA)과 조치 추적
  • 요구사항 기반 추적표로 제품 개발 누락 여부 확인

세부 기능 및 강점

  • 대량 엑셀 업로드 작업 가능
  • 업적자 배분으로 통합 진척 집계 가능
  • 서비스별 담당자별 현황 대시보드 제공
  • V&V 활동 관리로 SW 제품 품질 수준 인지
  • 품질 향상을 위한 방안 도출 가능

주요 메뉴 구성

SW Engineering(18)
  • 요구사항 정의/및 관리
  • 요구사항 등록, 상태, 변경(버전) 관리
  • 요구사항 Status 관리
  • 요구사항 간편 등록(By BGS)
  • 요구사항 추적관리
  • SW 기능 구현 및 UI 관리
  • SW 설계/프로세스 관리
  • SW 계획 및 설계검토 관리
  • SW 구성 관리 및 진척 관리
  • SW 구성 대가 및 실적 관리
Verification / SU Testing
  • 테스트 환경설정 및 준비
  • 테스트 Level별 설계 및 실행
  • 테스트 Level별 수행 및 진행추적
  • 통합테스트 운영
  • 결함 등록 및 실세 분석
  • 테스트 담당자별 진척현황(By 테스트)
